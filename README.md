# Solana Vault (Devnet)

A simple on-chain **Vault program on Solana Devnet** that allows users to **deposit** and **withdraw** funds securely.  
The project includes a smart contract written with the **Anchor Framework** and a **Next.js frontend demo** for interacting with the program.

## 🚀 Demo

Live Demo (Vercel):  
👉 **[https://solana-vault-demo.vercel.app/]**

> The demo frontend allows users to connect their wallet and interact with the Vault on **Solana Devnet**.

---

## ✨ Features

- Deposit SOL into the Vault
- Withdraw SOL from the Vault
- On-chain program deployed to **Solana Devnet**
- Web interface for interacting with the contract
- Wallet integration for user transactions

---

## 🛠 Tech Stack

**Blockchain / Smart Contract**

- Rust
- Anchor Framework
- Solana Program Library

**Frontend**

- Next.js
- Solana `web3.js`

**Deployment**

- Solana Devnet
- Vercel (Frontend Hosting)

## ⚙️ Local Development

### 1 Install Dependencies

Make sure you have installed:

- Rust
- Solana CLI
- Anchor Framework
- Node.js

### 2 Clone the Repository

```bash
git clone https://github.com/Jsirpro/Solana-vault-demo
cd Solana-vault-demo
```

### 3 Run the Frontend

```bash
cd app
npm install
npm run dev
```

---

## 🌐 Network

The program is deployed on:

**Solana Devnet**

Make sure your wallet is connected to Devnet when using the demo.
